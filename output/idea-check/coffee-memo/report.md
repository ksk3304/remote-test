## 5. 結論: 血圧・読書以外に勝ち筋のある記録系ニッチ

### 5.1 調査結果

**75点以上のニッチは見つからなかった**

調査した17ジャンル全てがKill判定となった理由：

1. **成熟市場**: ほぼ全ての記録系ジャンルで★4.0以上の定番アプリが存在
2. **大手参入**: 登山（YAMAP）、ゴルフ（楽天/GDO）等は企業が参入済み
3. **構造的不満の欠如**: 放置アプリがあっても、別の高品質競合がカバー済み

### 5.2 なぜ記録系は厳しいのか

記録系アプリには共通の問題がある：

| 問題 | 説明 |
|------|------|
| **差別化困難** | 「シンプル」「使いやすい」は既に実現済み |
| **乗り換えコスト** | 既存ユーザーはデータ移行を嫌う |
| **課金の壁** | 無料・低価格競合が多い |
| **DB構築の壁** | 専門ジャンルはDBが必要（香水、万年筆等） |

### 5.3 推奨: 記録系以外のアプローチ

記録系に固執するより、以下を検討すべき：

1. **ツール系**: 一回の利用で価値が完結するアプリ
2. **ユーティリティ系**: 特定の課題を解決するアプリ
3. **既存75点以上ニッチ**: 血圧（78.5点）、読書（77点）、習慣トラッカー（75.5点）、植物水やり（75点）
