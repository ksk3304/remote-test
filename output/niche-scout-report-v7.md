# ニッチ1: 広告ゼロ筋トレ記録アプリ（76点）【Go】

## 1. エグゼクティブサマリー

- **ニッチ**: 筋トレする人が、**広告ゼロ・データ消失なし・UI改悪なし**でシンプルにワークアウトを記録
- **判定**: **Go**
- **総合スコア**: 76点
- **Go理由**: 構造的不満が3種類成立（データ消失、広告問題、UI改悪）、Hevy以外の競合に明確な問題あり

## 2. 市場分析

### 2.1 ジャンル概要
筋トレ記録アプリは大きな市場。筋トレMemo、俺の筋トレ記録、Hevy、Strong、バーンフィットなど多数の競合が存在。しかし**筋トレMemoはデータ消失・広告・UI改悪の問題**、**俺の筋トレ記録は広告過多**という明確な構造的不満がある。Heavyは高品質だがSNS機能付き。

### 2.2 ニッチの定義

- **ターゲット**: ジムで筋トレする人（20-40代男性中心）で、広告なし・データ確実保存・シンプルUIを求める層
